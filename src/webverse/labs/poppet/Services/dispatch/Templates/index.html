Dispatch API

Order fulfillment API for the Poppet Shop. All endpoints require X-Api-Key header.

Orders

GET/api/v1/orders
List all fulfillment orders. Filter by status with ?status=
GET/api/v1/orders/{id}
Get order details by ID.
POST/api/v1/orders/{id}/pack
Mark order as packed and ready for shipment.
POST/api/v1/orders/{id}/ship
Create shipment with carrier and tracking number.

Shipments

GET/api/v1/shipments
List all shipments.
GET/api/v1/shipments/search?tracking={number}
Search shipments by tracking number.
GET/api/v1/shipments/{id}
Get shipment details.
GET/api/v1/shipments/{id}/track
Get tracking information for a shipment.

Carriers

GET/api/v1/carriers
List available shipping carriers.

Stats

GET/api/v1/stats
Fulfillment statistics and performance metrics.

Labels

POST/api/v1/labels/{order_id}
Generate a shipping label for an order.

Returns

GET/api/v1/returns
List return requests.
POST/api/v1/returns
Create a new return request.

Authentication

curl -H "X-Api-Key: YOUR_KEY" http://dispatch.poppet.local/api/v1/orders
