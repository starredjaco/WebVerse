Poppet Connect API

Integration reference for wholesale partners and internal services. Base URL: http://connect-api.poppet.local

List Products

GET/api/v1/products
Returns all products in the catalog.

Search Products

GET/api/v1/products/search?q={term}
Search products by name. Returns matching products as JSON.
